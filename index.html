Tentang Saya

Halo semua!!! Nama saya Amalia Maulina. Saya lahir di Bogor, 5 Juni 2001. Orang-orang memanggilku Mbul. Saya dipanggil Mbul dari SD. Saya dipanggil mbul karena saya suka ngemil. Sekarang saya tinggal di Ciomas, Bogor. Sekarang umurku 16 tahun, beberapa bulan lagi umurku 17 tahun, yeee.. Saya alumni SDN Taman Pagelaran, SMP Negeri 6 Bogor dan sekarang bersekolah di SMK SMAK Bogor kelas 11. Sewaktu saya kecil, saya memiliki banyak cita-cita, menjadi seorang dokter, astronot, dan quality control. Saat ini cita-citaku nmenjadi seorang quality control, cocok dengan jurusanku yaitu Analisis Kimia. Saya berharap bisa menjadi quality control di perusahaan yang besar. Setelah lulus dari SMAKBO, saya ingin melanjutkan kuliah disalah satu universitas di Bandung, tetapi orang tuaku menyuruhku untuk langsung bekerja.

Saya memiliki banyak hobi, seperti berenang, membaca komik, kulinerm dan jalan-jalan, kuliner dan membaca buku. Dulu saya takut dengan air, tetapi semenjak saya les berenang, saya menjadi tidak takut air. Sebelum sekolah di SMAKBO, saya sering sekali berenang. Tetapi semenjak sekolah di SMAKBO, saya tidak memiliki waktu untuk berenang. Saya suka membaca berbagai macam buku, seperti komik dan novel. Saya kurang suka membaca buku pelajaran, karena kalau saya membaca buku pelajaran, saya akan pusing sendiri. Saya juga kurang suka membaca buku yang tidak seru. Saya suka membaca buku yang menyenangkan yang membuat saya terhibur.

Saya suka sekali jalan-jalan bersama keluarga saya, walaupun jarak jalan-jalannya dekat, asalkan bersama keluarga saya tetap menyukainya. Hampir setiap minggu kami jalan-jalan disekitar Bogor. Kami memilih jalan-jalan disekitar Bogor, karena kami tidak memiliki waktu untuk pergi ke luar kota. Saya suka jalan-jalan ke tempat alam seperti Panorama Pabangbong, Bukit Geulis, dan sebagainya. Saya juga suka jalan-jalan ke tempat wahana-wahana permainan seperti Jungle Land, Jungle Fest, dan sebagainya. Akhir-akhir ini saya jarang jalan-jalan bersama keluarga saya dikarenakan kesibukan masing-masing :(.
